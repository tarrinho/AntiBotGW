## Changes in this version

- **agents-bucket `gwmgmt` key** — `agents_bucket_detail_endpoint` now includes a `gwmgmt` counter reflecting admin-namespace (`/antibot-appsec-gateway/`) events within the time bucket.
- **`_serve_mirrored_404` crash fix** — guarded against `KeyError` when `_upstream_404_cache` is empty (cold-start race).
- **UPSTREAM hot-reload flushes 404 cache** — when `UPSTREAM` changes via `/secured/config`, the cached 404 body is cleared so the next admin-blocked request fetches from the new upstream.
- **`BYPASS_PATHS` hot-reload knob** — comma-separated path prefixes that skip all detection; configurable at runtime via `POST /secured/config`.
- **`JS_CHAL_OPEN_PATHS` hot-reload knob** — comma-separated paths exempted from the JS challenge gate; configurable at runtime.
- **`bump-version.sh`** — new shell script atomically updates all canonical version locations across the repo (config.py, test_pure.py stale-string regex, proxy.py docstring, docker-compose, all dashboards, README quickstart, test_geo_dashboard.py).
- **`tests/test_endpoints_dynamic.py`** — 114-test live aiohttp `TestServer` + `TestClient` integration suite covering all admin HTTP endpoints and cache-control headers (ported from v1.7.9 session).
- **`status_endpoint` Cache-Control fix** — `core/proxy_handler.py` `status_endpoint` now returns `Cache-Control: no-store` (was the only admin endpoint missing it).
- **`rules.md` §13a bump-version step** — `bump-version.sh OLD NEW` added as Step 0 to the version consistency review section.
- **H5: unbounded dict growth fixed** — four dicts capped: `_ACTIVE_SESSIONS` (evicted in `_prune_state_loop` step 10 using wall clock `_time.time()`), `_signal_order_cache` (capped 2000 → 1000 in step 10), `_asn_path_clusters` (evict entries older than 10 min in step 10), `_LOGIN_BUCKET` (inline O(n) eviction inside `_LOGIN_BUCKET_LOCK` on every call to `_login_rate_limit()`).
- **M2: dead duplicate try/except removed** — `scoring._load_signal_order_cache()` and `scoring._save_signal_order()` each had an unreachable nested try/except block for `from admin.mesh import _gw_local_id`; collapsed to a single try/except.
- **`LABYRINTH_LINKS_PER` knob name fix** — `_HOT_RELOAD_KNOBS` had `LABYRINTH_LINKS_PER_PAGE` (the env-var name) instead of `LABYRINTH_LINKS_PER` (the Python variable name); `_read_hot_reload_state()` silently skipped it, causing it to never appear in `GET /config` or exports.
- **`tests/test_v1711.py`** — 13 new static QA tests for H5 + M2 (prune logic, login bucket eviction, dead-code removal).
- **`tests/test_h5_m2_dynamic.py`** — 18 new dynamic tests running real coroutines: `_prune_state_loop` via patched `asyncio.sleep`, `_login_rate_limit` end-to-end, `_load_signal_order_cache` / `_save_signal_order` with stubbed `admin.mesh`.
- **`tests/test_settings_config_functional.py`** — 49-test functional suite for settings export/import and `GET/POST /secured/config`; includes 6-test `TestSettingsImportEnvPinned` class verified against live dynamic test.
