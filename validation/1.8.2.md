Notes:
- All SQL in `analytics.py` uses parameterised queries (`conn.execute(sql, (start_b, end_b, bucket_secs))`) — no string formatting into SQL
- `_percentile()` is a pure-math helper with no I/O; input list is always pre-sorted from the caller
- `loadVhostHeatmap()` and `loadTrafficChart()` pass `range`/`bucket`/`end` via `URLSearchParams` in GET query params — no XSS path as values are integers from `<select>` DOM controls
- Heatmap HTML table generation uses `textContent` assignment, not `innerHTML` concatenation — no XSS risk
