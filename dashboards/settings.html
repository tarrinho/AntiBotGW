Gateway Registry

Manage every gateway in the distributed mesh: keypair lifecycle, per-pair distribution topology, and an append-only audit trail of every registry mutation. Each gateway holds a unique gw_id, an HMAC keypair (private secret + SHA256 verification handle), a coarse can_distribute flag, and a directional sync rule set. Mesh storage is Redis (when REDIS_URL is set); registry of record is SQLite (mirrored to Postgres).
